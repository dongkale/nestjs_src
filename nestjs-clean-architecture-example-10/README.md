## Description

Starter [Nest.js](https://github.com/nestjs/nest) in Clean Architecture and TypeScript.
Just a basic Todo task app example.

## Swagger
Swagger is available at : http://localhost:3000/api-docs

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
