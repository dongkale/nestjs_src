## Description
  Base application API made with NestJS, TypeORM, GraphQL, and Jest

###  Folder structure
    Code organization based on NestJS modules with Domain Driven Design, focused on codebase scalability.

### SOLID
    Using SOLID principles to provide better code design for easier maintenance and testing.

### GraphQL / REST
    You can either create API's REST or Graphql

## Installation

```bash
$ npm install
$ docker-compose up -d
$ cp .env-sample .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Endpoints
```
/users or /graphql
```

## Swagger documentation
```
/docs
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
