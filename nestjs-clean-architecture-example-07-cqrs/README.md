<p align="center">An implementation of Clean Architecture and CQRS for NestJS.</p>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Documentation

```bash
# open on the web browser
$ http://localhost:3001/api
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
